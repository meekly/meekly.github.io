# Valentin Kiselev - Portfolio

Personal portfolio website showcasing software engineering projects and skills.

## Local Development

To run the portfolio locally:

```bash
# Install dependencies
npm install

# Start the local server
npm start
```

The site will be available at `http://localhost:3000`

## Project Structure

- `index.html` - Main portfolio page
- `server.js` - Simple HTTP server for local development
- `cart/` - Interactive shopping cart demo
- `you/` - YouTube search application
- `frmw_*/` - Framework demonstration projects

## Technologies

- Go, Ruby, JavaScript/TypeScript
- React, Vue.js
- Node.js for local development
